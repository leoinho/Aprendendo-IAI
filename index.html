Olá! Este site é para alunos e não alunos de computação, que tenha algum tipo de dificuldade com a matéria de Introdução a informática. Para melhor aprendizado do aluno, esta matéria será dividida em dois módulos, exatamente como é dividida para os alunos que fazem computação pelo CEDERJ. Porém, vamos abordar o assunto de uma forma mais dinâmica, de um modo que seja mais fácil o aprendizado do aluno.
